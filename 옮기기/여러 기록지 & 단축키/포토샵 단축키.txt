** 포토샵 단축키

Alt + 마우스 휠 조작 : 크기 조정
Ctrl + + : 화면 크게
Ctrl + - : 화면 작게
Ctrl + 0 : 화면 꽉차게
Ctrl + 1 : 100%로 보기

Ctrl + D : 선택 영역 해제

* Ctrl + J
1. 선택 영역이 있는 상태에서 Ctrl + J : 선택영역을 복사
2. 선택 영역이 없는 상태에서 Ctrl + J : 레이어 복제
3. 레이어로 새로 생성

Ctrl + G : 그룹핑

Alt + Delete : 앞의 색 채우기
Ctrl + Delete : 뒤의 색 채우기

Ctrl + T : 자유 변형툴(이동, 크기조정, 회전)

Ctrl + 썸네일 클릭 : 선택 영역화

Ctrl + E : 선택한 레이어 합치기

[ : 브러쉬 사이즈 작게
] : 브러쉬 사이즈 크게

Ctrl + Alt + G : 클리핑 마스크 적용/해제
Ctrl + E : 레이어 합치기
